# 🔥 火凤论坛

一个现代化的综合讨论社区，基于纯静态网页实现，使用 localStorage 模拟数据存储。

---

## 📖 关于论坛

火凤论坛是一个多板块的综合讨论平台，支持：

- 💬 **技术讨论** - 编程、硬件、人工智能等
- 📈 **股票金融** - 市场分析、投资交流
- ☕ **生活闲聊** - 日常话题、兴趣爱好
- ❓ **问答求助** - 问题解答、经验分享
- 📰 **新闻资讯** - 时事热点、行业动态

---

## ✨ 主要功能

✅ 多板块分类浏览  
✅ 发帖与评论系统  
✅ 帖子点赞（赞同）  
✅ 标签系统与热门标签  
✅ 响应式设计（支持手机/平板/桌面）  
✅ 深色主题界面  
✅ 数据本地持久化（localStorage）

---

## 🛠️ 技术实现

- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **存储**: LocalStorage（模拟后端数据库）
- **设计**: 深度暗色主题，渐变紫色主色调
- **布局**: CSS Grid + Flexbox 响应式布局
- **交互**: 纯前端实现，无需后端

## 🚀 使用说明

1. 直接打开 `index.html` 即可使用
2. 支持所有现代浏览器（Chrome/Firefox/Safari/Edge）
3. 数据保存在浏览器本地，清除缓存会丢失数据
4. 点击左侧导航栏切换板块
5. 点击"📝 发帖"按钮发布新帖
6. 点击任何帖子查看详情并评论

---

## 🎨 界面预览

论坛采用 Reddit/贴吧风格设计：

- **左侧导航**：板块分类切换
- **中间内容区**：帖子列表/详情
- **右侧小工具**：热门标签、论坛统计、规则提示
- **顶部栏**：Logo、发帖按钮、主题切换
