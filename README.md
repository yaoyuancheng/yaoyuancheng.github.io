# ⚛️ 量子力学简介

这是一个介绍量子力学的静态网页项目，用于科普和学习。

---

## 📖 关于

本网页系统介绍量子力学的基本概念、历史发展、核心原理和现代应用。

- **主题**: 量子物理学基础
- **目标**: 让更多人了解量子力学的奇妙世界
- **状态**: 持续更新中 🚀

---

## 🔬 内容概览

| 模块 | 说明 |
|------|------|
| 🌊 波粒二象性 | 粒子既是波又是粒子 |
| 🎲 不确定性原理 | 位置与动量无法同时精确测量 |
| 🔀 量子叠加 | 系统可同时处于多个状态 |
| 👥 量子纠缠 | 超距作用的鬼魅现象 |
| 💻 量子计算 | 未来的计算革命 |
| 🔐 量子通信 | 绝对安全的加密方式 |

---

## 🛠️ 技术栈

- **前端**: HTML5, CSS3 (渐变、动画、响应式)
- **设计**: 深色主题，配以量子蓝/紫色的渐变
- **特性**: 现代化卡片布局、hover 动效、移动端适配
